# Fizzy Pins Collapsed

Image file: `fizzy-pins-collapsed.png`

This screenshot shows Fizzy's pinned tray in its collapsed state. A single pinned
card preview sits over the bottom-left nav area, with only a small portion of
stacked cards showing behind it. The preview keeps the same left-column width as
the Pinned nav item and visually covers that nav item while collapsed.

Future agents should interpret this as the target collapsed behavior for
Trojes' desktop pinned tray: same width as the Pin control, taller than the nav
bar, card-forward, and intentionally covering the Pin button until expanded.
