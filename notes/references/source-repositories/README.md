# Source Repository References

Store source repository references as notes and URLs only. Do not commit cloned
repositories here.

For each repository reference, include:

- Source URL
- Why it matters for Trojes
- What files, flows, or implementation ideas are worth inspecting
- Any known caveats or context needed before cloning

When inspection is needed, clone the repository into a disposable location
outside the project unless the user explicitly asks for a checked-in copy.
